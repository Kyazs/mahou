# Product

## Register

brand

## Users

opencode power users — developers already using (or evaluating) opencode who want disciplined engineering workflows without plugins, MCP servers, or npm dependencies. They arrive from GitHub, CLI communities, or word of mouth, often mid-session or mid-project, and need a fast path from “what is this?” to install.

Primary job: understand mahou’s promise in under a minute, trust the craft, and install or clone.

## Product Purpose

mahou is a portable set of opencode slash commands that enforce disciplined software engineering — root cause before fix, design before code, verify before ship. The landing page is the brand surface: it sells the philosophy and the install path, not a product dashboard.

Success looks like: a visitor leaves with “discipline as magic” as a felt idea, clear install next steps, and enough credibility to star, clone, or run the installer.

## Brand Personality

Dark · Ritual · Mythic, with ink-craft precision underneath.

Voice is confident and exact — ceremonial without fluff. Emotion goal: **mystical power** — the sense that disciplined process is a kind of magic you can wield, not a checklist you endure. Visual language is black lacquer + cinnabar seal + aged brass (not violet spirit glow, not neon cyberpunk).

## Anti-references

- Generic SaaS purple: gradient blobs, identical feature-card grids, startup clichés
- Corporate enterprise: navy sterility, stock-photo trust theater, compliance tone
- Neon cyberpunk overload: glitch spam, matrix rain, glow-for-glow’s-sake
- Minimal empty portfolio: sparse whitespace with nothing to say

## Design Principles

1. **Discipline is the magic** — show order, sequence, and craft as the source of power, not decorative sparkle alone.
2. **Ceremony over chrome** — every motif (kanji, ink, seal, wave) must earn its place; no motif without meaning.
3. **Install is the climax** — narrative builds toward a clear, frictionless install path for opencode users.
4. **Mythic, not kitsch** — Japanese cues are sumi-e / ritual craft, never anime cosplay or tourist Japan.
5. **Power with restraint** — bold electric moments on a dark field; quiet exactness in body and structure.

## Accessibility & Inclusion

WCAG AA as the bar: body text contrast ≥4.5:1, large text ≥3:1, visible focus states, and full `prefers-reduced-motion` alternatives for all motion. Dramatic dark UI is allowed; contrast is not optional.
